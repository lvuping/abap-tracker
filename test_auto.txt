Test cases for auto.ahk:

Test 1: Basic sequence processing
Input: N 2024.11.15 김동현 Some change description
Expected: U1 2025.09.10 DH2025.KIM SAP ID replacement

Test 2: With U numbers
Input: U1 U2 U3 2024-11-15 박지민 Another change

Test 3: With C-prefixed ID (should be removed)
Input: N C2001023213 2024/11/15 이서준 Description text here
Expected: U1 2025/09/10 DH2025.KIM SAP ID replacement

Test 4: Complex case
Input: U1 C123456 2024.11.15 홍길동 This is the change description
Expected: U2 2025.09.10 DH2025.KIM SAP ID replacement

Test 5: No sequence number (should add N)
Input: 2024-11-15 김철수 Some text

Usage:
1. Run the auto.ahk script
2. Select a test input line
3. Press F1
4. The InputBox will show the processed text
5. Click OK to replace the selected text
